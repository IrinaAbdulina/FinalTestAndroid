
Мобильная автоматизация 38 модуль

1. Всего написано 14 тестов. 11 тестов позитивные, 3 теста негативные на проверку локатора и элемента.
2. Поиск вынесен в отделый метод с возможностью передачи типа поиска, либо по части текста найденого контента, либо по полному тексту xpach найденных элементов.

3. Содержимое репозитория:
pom-xml - основной конфиг Maven
apps/ - приложение Википедии для тестирования
src/test/java/tests/ - директория с тестами
src/test/java/lib/ - директория со вспомогательными классами -- CoreTestCase.java - родительыский класс для тестов
src/test/java/lib/ui/ - директория с PageObject-классами -- PlatformSelector.java - класс с указанием платформы для запуска
src/test/java/lib/ui/pages/ - директория в PageObject-классами
